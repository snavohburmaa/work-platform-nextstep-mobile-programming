# NextStep - Social Job Platform

A modern Flutter social platform where everyone can post jobs and everyone can apply - connecting job seekers and employers in one beautiful app!

## Features

- ✅ User registration and login
- ✅ Post job opportunities
- ✅ Apply to jobs (except your own posts)
- ✅ View user profiles
- ✅ Direct messaging between users
- ✅ Track your applications
- ✅ Manage your job posts

## Technology Stack

- Flutter (Dart)
- SharedPreferences (Local storage)
- Material Design

## Getting Started

```bash
# Install dependencies
flutter pub get

# Run the app
flutter run
```

## Project Structure

```
lib/
├── constants/       # App colors
├── themes/          # App theme configuration
├── models/          # Data models
├── services/        # Storage service
├── pages/           # All app screens
└── widgets/         # Reusable widgets
```
